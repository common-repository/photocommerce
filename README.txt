Connect the PhotoCommerce app, which allows you to easily upload photos to your WooCommerce products and product variations, as well as edit the price and title

== Description ==

PhotoCommerce is a powerful plugin that allows you to easily upload pictures to your WordPress website and edit products directly from your mobile device using the PhotoCommerce mobile app. With PhotoCommerce, you can manage your online store on-the-go, from anywhere in the world.

To get started, simply activate the PhotoCommerce plugin on your WordPress website, and generate an application password on the setup page. Then, download the PhotoCommerce mobile app from the App Store or Google Play, and connect it to your WordPress website using the application password.

PhotoCommerce is designed to be user-friendly and easy to use, even for non-technical users. It's a secure and reliable solution for managing your online store from anywhere, with complete control over your product information and images.

**It's important to note that the plugin only sends product info changes and new images to your WordPress site. No data is sent anywhere except to your servers, ensuring that your data is kept private and secure. If you have any questions or issues, the PhotoCommerce support team is always available to help.**

== Installation ==

IMPORTANT: The logged in user must be an admin user with read and write permissions in order to use this plugin.

1. Download the PhotoCommerce plugin from your account on the WordPress website or through the WordPress plugin directory.
2. Log in to your WordPress website's dashboard and go to the "Plugins" section.
3. Click the "Add New" button and upload the PhotoCommerce plugin.
4. Once the plugin is uploaded and installed, activate it.
5. After activation, you'll be redirected to the setup page. Here you can generate a new application password to connect the mobile app.
6. Click on the **"Generate Application Password"** button, and a new password will be generated.
7. A QR code and manual code will appear on the page. Use the QR code or manual code to connect the mobile app to your WordPress website.
8. Download and install the PhotoCommerce mobile app from the App Store or Google Play.
9. Open the app and click on the "Add Connector" -> "Connect WooCommerce" button.
10. Follow the steps and scan the QR code or enter the manual code to connect the app to your WordPress website.
11. Once connected, you can easily upload pictures and edit products from the mobile app.
12. **It's important to note that all data is sent only to your own WordPress website/server and nowhere else. Your data is kept secure and private.**

That's it! Your PhotoCommerce plugin is now installed and connected to the mobile app. Enjoy the convenience of managing your WordPress website from anywhere with the PhotoCommerce mobile app.

== Frequently Asked Questions ==

= How do I get the Application Password to connect the PhotoCommerce mobile app to my WordPress website? =
To get the Application Password, activate the PhotoCommerce plugin on your WordPress website and go to the setup page. On the setup page, click the "Generate Application Password" button. A new password will be generated, and you'll see a QR code and manual code. You can use either the QR code or the manual code to connect the mobile app to your WordPress website. Just scan the QR code with the app or enter the manual code in the app to connect. If you have any issues, the PhotoCommerce support team is available to help.

= Can I use the PhotoCommerce mobile app to upload pictures to my website from anywhere in the world? =
Yes, as long as you have an internet connection, you can use the PhotoCommerce mobile app to upload pictures to your website from anywhere in the world.

= How secure is the data that is sent to my WordPress website/server from the mobile app? =
The data that is sent to your WordPress website/server from the mobile app is completely secure and private. The connection is encrypted and your data is not shared with any third-party services.

= Can I use the PhotoCommerce mobile app to manage other aspects of my WordPress website, like blog posts or page content? =
No, the PhotoCommerce mobile app is specifically designed for managing and editing products. If you want to manage other aspects of your WordPress website, you will need to use the WordPress dashboard.

= Do I need any specific technical knowledge or expertise to install and use the PhotoCommerce plugin? =
No, you don't need any special technical knowledge or expertise to install or use the PhotoCommerce plugin. The installation process is straightforward, and the plugin is designed to be user-friendly and easy to use, even for non-technical users. If you do run into any issues or have questions, the PhotoCommerce support team is available to assist you.
